# cloud-api-commonts 
该模块是一个通用模块可以把通用的东西放在这里
1.pom.xml 文件可以通过下边的方式引入：
     <!--引入自己定义的jar  通用包 -->
            <dependency>
                <groupId>com.bank.springcloud</groupId>
                <artifactId>cloud-api-commonts</artifactId>
                <version>${project.version}</version>
            </dependency>
